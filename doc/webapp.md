# vcf-explorer web app

## Requirements
- [nodejs](http://nodejs.org/)
- [bower](http://bower.io/)
- [grunt](http://gruntjs.com/)

### Install development environment

```sh
  cd /path/to/vcf-explorer/src
  npm install
  bower install
```

### Run the development server

```sh
  cd /path/to/vcf-explorer/src
  grunt serve
```
